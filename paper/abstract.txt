Williams (STOC 2025) proved that every multitape Turing machine running in time t can be simulated in space O(sqrt(t log t)), via Tree Evaluation and the Cook-Mertz algorithm. We transfer the same block-size optimization to the dominant sequential memory structure in LLM serving: the key-value (KV) cache of length S. We do not claim that Transformers are multitape TMs. Setting checkpoint stride b ~ sqrt(S log S) and retaining only boundary checkpoints plus one active block (sqrt(S) residency) reduces measured KV storage by 90-97% for S in [1k, 16k] across real models from 135M (Apple Silicon) to 14B (Modal NVIDIA A100), while host-paged online-softmax attention remains numerically exact. Naive whole-prefix recompute each token increases peak device memory. Code and JSON benchmarks: github.com/gftdcojp/cloud-murakumo.
